# HAMI
Implementation of Hippocampal-Augmented Memory Integration
This repository provides the reference notebook for the HAMI (Hippocampal-Augmented Memory Integration) framework and the HiCoS (Hippocampal Context-dependent Sequences) environment used in the paper "A scalable reinforcement learning framework inspired by hippocampal memory mechanisms for efficient contextual and sequential decision making".

[HiCoS_Final_NF.pdf](https://github.com/user-attachments/files/24998048/HiCoS_Final_NF.pdf)




Reference

Poursiami H, Moshruba A, Cooper KW, Gobin D, Kaiser MA, Singh A, Noor R, Shahbaba B, Jaiswal A, Fortin NJ, Parsa M. [A scalable reinforcement learning framework inspired by hippocampal memory mechanisms for efficient contextual and sequential decision making.](https://doi.org/10.1038/s41598-025-10586-x) Scientific Reports. 2025;15(1):25221.
